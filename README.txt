Cos Lettuce Growth Stage Image Dataset
================================================================

DESCRIPTION
-----------
This dataset contains images of Cos lettuce (Lactuca sativa var. longifolia)
collected across three growth stages: small (seedling), medium (mid-growth),
and large (near-harvest). Images were collected from a controlled indoor
farming system and from Hydro Green Garden Farm, with the farm's consent.
The dataset was used to train and evaluate a YOLOv11 object detection model
for plant growth-stage classification and computer-vision browning detection,
as described in the associated publication.

ASSOCIATED PUBLICATION
----------------------
Chaoraingern, J., Pattaraanuvong, A., Paraksa, K., Khunthong, K.,
Nontiwantok, T., and Numsomran, A.
"Agentic AI-Driven Cultivation Advisory and Plant Disease Diagnosis
in a Controlled Indoor Farming System."
AgriEngineering (MDPI), 2026. [DOI to be assigned]

DATASET CONTENTS
----------------
Total images : 301
Format       : JPEG (.jpg)

Folder structure:
  /small/    102 images  (seedling stage)
  /medium/    99 images  (mid-growth stage)
  /large/    100 images  (near-harvest stage)

Note: Bounding-box annotations are not included.
Researchers wishing to use this dataset for object detection
are encouraged to annotate using tools such as Roboflow or CVAT.

DATA COLLECTION
---------------
-- Field images: Hydro Green Garden Farm (collected with consent)
